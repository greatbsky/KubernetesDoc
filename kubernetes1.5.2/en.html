Installation of kubernetes v1.5.2 step by step

Kubernetes is an open-source system for automating deployment, scaling, and management of containerized applications.

It groups containers that make up an application into logical units for easy management and discovery. Kubernetes builds upon 15 years of experience of running production workloads at Google, combined with best-of-breed ideas and practices from the community.

Recently there is a new project need to deploy, compared openstack, for the lack of hardware resources, I choose container for better performance, and I need to deploy, manage it easier, so I decide to set up a kubernetes cluster.

This public doc is my note in develop environment, not production environment. I have ever redone some times so maybe there are some mistake.
